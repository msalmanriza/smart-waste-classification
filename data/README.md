# Dataset

Dataset yang digunakan dari project ini adalah Garbage Classification Dataset dari Kaggle.

Link Dataset:
https://www.kaggle.com/datasets/mostafaabla/garbage-classification

Struktur folder:

- raw/ : menyimpan dataset asli hasil download dari Kaggle.
- processed/ : menyimpan dataset yang telah dipisahkan menjadi train, validation, dan test.

Dataset tidak disertakan dalam repository karena ukuran file yang besar.
